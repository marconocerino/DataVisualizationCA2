Dynamic / Interactive Features Added

  • Modules and Grade updates in the Education Section: Selecting a year in the dropdown updates the module grades table dynamically (education.js).
  • Animations when user is scrolling the website: The cards in the website have a revealing animation when scrolling (scrollEffect.js).
  • Contact modal : Opens from multiple buttons (“Get in Touch”, “Say hi!”, and in the Footer “Get in touch”).
  • Filters in the Project Section: The filter buttons (All/Web/Python/Java) show/hide project cards (projectsFilter.js).
  • Responsive mobile: On small screens the navbar collapses into a toggler button and expands to show links, the font sizes and layouts are automatically adjusted depending on the screen sizes.
  • Navigation bar highlighting: The navbar updates the active button in the navigation bar while scrolling.
